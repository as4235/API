# Coo-e Management App

## Prerequisites

cd to directory `nextjs`
Create a .env file in project root folder and add the following content (fill in the blanks)

```
AUTH0_SECRET=
AUTH0_BASE_URL=
AUTH0_ISSUER_BASE_URL=
AUTH0_CLIENT_ID=
AUTH0_CLIENT_SECRET=
```

## To run the app locally

```
npm install
npm run dev
```
